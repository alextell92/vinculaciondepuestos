Sistema de Evaluación y Jerarquización de Puestos


Este sistema tiene como objetivo garantizar una estructura salarial equitativa y alineada con el mercado. A través de una serie de evaluaciones, se asignan puntos a factores clave de cada puesto, lo que nos permite tomar decisiones informadas sobre la jerarquía salarial.

Escalas de puntuación

El sistema de evaluación de puestos se basa en la asignación de puntos a distintos factores clave que definen el valor de cada puesto dentro de la organización.
